проанализируй мой проект начни с точки входа в приложение и session-manager. в проекте
bot-missions
должен собирать актуальную информацию о миссиях игрока и сохранять ее для различных ответных действий. Нужно чтоб бот умел отличать различные типы миссий. например экспедиции, атаки чужих флотов, транспорт своих кораблей.
первая задача это научиться пересоздавать экспедицию по возвращению. для этого нужно отслеживать все экспедиции их координаты и время возврата. с небольшой задержкой в 1-5 минут переходить на ту луну с которой экспедиция завершилась и отправлять весь доступный на луне флот всех типов кораблей в новую экспедицию по тем же координатам. Например если луна по координатам 1:910:5 тогда экспедиция с этой луны должна быть в 1:910:16. важно чтоб все 5 доступных экспедиции перезапускались по кругу. если все 5 экспедиций в полете то бот ответственных за экспедицию просто ожидает.
для выполнения задания тебе нужно запустить игру и проанализировать поля-таблицы доступные в браузере и их содержимое. так же проанализировать какими кнопками открываются меню и отправка мисии. при необходимости вывести в терминал некоторую информацию для понимания работы сайта и потом уже реализовывать задачу опираясь на правильные данные. логика отслеживания разных миссий должна ббыть в bot-missions. тут только контроль за миссиями, а
вся логика старта каждой отдельной экспедиции - выбор флота и цель мисии должна быть в expedition. в дальнейшем может реализуем миссии с транспортом или другие в отдельных файлах.
